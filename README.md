# Flower Shop

## Небольшая презентация проекта

Это итоговая работа по курсу программирования. Проект сделан как сайт цветочного магазина: есть каталог букетов, корзина, избранное, форма контактов, оформление заказа и простая админ-панель для просмотра заявок.

Сайт написан на PHP, HTML, CSS и немного JavaScript. Данные о товарах хранятся в отдельном PHP-файле, а заявки сохраняются в базу данных MySQL. Верстка сделана в спокойном розовом стиле, с фоновыми изображениями и карточками товаров.

## Что есть на сайте

- главная страница с кратким описанием магазина;
- каталог цветов с поиском;
- карточки букетов с названием, ценой, корзиной и избранным;
- wishlist, откуда товар можно перенести в корзину;
- корзина с изменением количества;
- форма оформления заказа в корзине;
- страница контактов с формой сообщения;
- отдельная страница FAQ;
- простая админ-панель без авторизации, где видны заявки.

## Как мы делали проект

Сначала собрали обычные страницы сайта, потом постепенно добавляли рабочую логику. Каталог и корзина работают через сессии, чтобы можно было добавлять товары без сложной системы пользователей.

Добавили отдельную отправку в MySQL, потому что для этого проекта надежнее хранить все заявки там и смотреть их через админ-панель.

Верстку дорабатывала постепенно: выравнивала карточки, добавляла новые изображения букетов, настраивала wishlist, корзину, FAQ и внешний вид кнопок.

## База данных

Для работы заявок нужна база MySQL. Таблица создается из файла:

```text
data/schema.sql
```

Настройки подключения находятся здесь:

```text
includes/db.php
```

Нужно указать:

```text
DB_HOST
DB_NAME
DB_USER
DB_PASS
```

## Какие данные сохраняются

Сайт сохраняет только данные, которые нужны для заявки:

- имя;
- email;
- телефон;
- адрес;
- сообщение;
- товары из корзины;
- сумма заказа;
- стоимость доставки;
- итоговая сумма.

IP, user agent и лишние технические данные не собираются.
